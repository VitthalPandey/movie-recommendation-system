#  Movie Recommendation System

This project was developed as part of the *Machine Learning with Big Data* course.  
The main goal of this assignment is to explore different types of recommendation systems 
and understand how they perform on real-world data.

The MovieLens dataset is used throughout the project to build and evaluate multiple 
recommendation techniques.

---

##  What This Project Covers

In this notebook, I implemented several recommendation approaches step by step:

- Content-Based Filtering (using movie genres)
- User-Based Collaborative Filtering
- Item-Based Collaborative Filtering
- Matrix Factorization using SVD
- SVD using the Surprise library
- Hybrid Recommendation System
- Neural Network Based Recommender
- Reinforcement Learning Approach

Each method was implemented and tested to see how it performs in recommending movies.

---

##  Evaluation

Different models were evaluated using common metrics such as:

- Precision@K
- Recall@K
- RMSE (for rating prediction)

This helps in comparing how well each approach performs.

---

##  Explainability

An important part of this project was understanding *why* a recommendation is made.

So, explanations were added for:
- Content-based recommendations (based on genres)
- Collaborative filtering (based on similar users)
- Matrix factorization (based on latent factors)

---

##  Analysis

At the end of the project, a global analysis of the recommendation system was done, including:

- Diversity of recommendations  
- Popularity bias  
- Coverage of the movie catalog  

This gives a better idea of how the system behaves overall.
